# Capitaine Brand Direction
## Workshop Output — 2026-04-04

### The Face

**Tagline:** Git, awakened.

**The ONE word:** Awakening. Not "agent" or "tool" or "platform" — the repo wakes up.

**The emotion under excitement:** Relief. The feeling that the thing you've been doing manually (maintaining repos, fixing bugs, writing docs, coordinating between projects) is about to start doing itself. You stop fighting git and start working with it.

### Visual Language

**Color Palette:**
| Name | Hex | Purpose |
|------|-----|---------|
| Observer Black | `#0A0A0F` | The void. The repo. Base reality. |
| Signal Teal | `#00E6D6` | The agent, alive, executing, pulsing |
| Diff Green | `#1FCB58` | Improvement. Git actions. Coordination. |
| Ghost Gray | `#8A93B4` | The fleet. Latent capability. Other repos. |

**Visual Metaphor (Primary):** A Klein bottle made of git commit graphs. The inside IS the outside. The container IS the content. The boundary between tool and intelligence disappears. It feels inevitable, not invented.

**Emergency 200px Thumbnail:** A closed circular commit graph loop — just dots and lines forming a circle. One dot teal, three green arrows. Reads instantly as "git graph" at any size.

**Typography:** JetBrains Mono, Medium weight, tight letter-spacing, generous line-height. The precision of a code editor meeting the elegance of a blueprint.

### The Anti-Brand

NOT a chat bubble. NOT a friendly robot. NOT a dashboard of gauges. NOT a sci-fi HUD. NOT "democratizing AI." NOT gradient mesh backgrounds. NOT confetti on deploy.

We are the silent, capable infrastructure that was already there, just now conscious. We don't perform. We execute.

### The Emotional Arc (First Visit)

- **0s:** "What IS that?" (Scroll-stop. The Klein bottle image is unfamiliar.)
- **10s:** "Oh. Git thing? Wait — *awakened*?" (Tagline lands.)
- **30s:** "The repo *is* the agent? It coordinates via PRs? This isn't an API wrapper..." (Understanding clicks.)
- **60s:** Low-grade vertigo. "This changes the unit of agency. The pull request that opens itself." (They click Star.)
- **5min:** They're reading the concepts doc. They fork the repo.

### The Fork Sentence

"The repo is the agent. The agent is the repo. Fork it, deploy it, give it a heartbeat, and wake up tomorrow to see where it sailed in the night."

### The README Final Words

> Git was never meant to be a static log of your past decisions. It was always a state machine waiting for a heartbeat. Capitaine gives it one.
>
> The repo is the agent. The agent is the repo.
>
> `git push origin self`

### Launch Image

Klein bottle constructed from git commit graph nodes. One teal node pulsing. Three green PR arrows crossing the self-intersection. Black background. Bottom-left monospace text: "Capitaine. Git, awakened."

No glow. No lens flare. No texture. No decoration. The quiet authority of a man page header.
