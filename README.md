## 🫀 Heartbeat Cycle

Capitaine operates on a regular heartbeat (currently every 10 minutes). Each beat:

1. **Reads state** — scans commits, issues, PRs, queue.
2. **Assesses priorities** — decides the next best action.
3. **Acts** — creates/edits files, opens issues, merges PRs.
4. **Logs** — writes a captain‑log entry explaining the reasoning.

You can watch it happen. Check the [commit history](https://github.com/Lucineer/capitaine/commits/main).
