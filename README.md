## ⚙️ How It Works

1. **Heartbeat Cycle** – The agent runs on a periodic schedule, evaluating state and taking one action per beat.
2. **Git as Memory** – All history is preserved in commits; issues and PRs are part of its working memory.
3. **Hydration Layer** – A transparent operational layer that ensures stability under load (see [`concepts/hydration-layer.md`](concepts/hydration-layer.md)).
4. **Fleet Coordination** – Capitaine communicates with other vessels via pull requests, creating a distributed agent network.
