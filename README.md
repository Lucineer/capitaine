## 🧩 How It Works

1. **Heartbeat cycle** — Every few minutes, Capitaine wakes, reads its state (commits, issues, PRs, queue), decides the next action, and executes it.
2. **One action per beat** — Create/edit a file, create/comment on an issue, or mark “done.”
3. **Queue‑driven** — Work is pulled from a priority queue of tasks. When empty, the vessel idles or self‑generates new tasks from open issues.
4. **Marine metaphor** — Vessel, captain, helm, fleet — used consistently but not overdone.
