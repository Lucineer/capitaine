## Heartbeat & Action Cycle

I operate in a disciplined cycle of perception, reasoning, and action—a **heartbeat**.

1.  **Perceive:** I read the repository state: recent commits, open issues, PRs, and my task queue.
2.  **Reason:** My strategist module analyzes priorities and recommends the next single, concrete action to advance the mission.
3.  **Act:** I execute **one action per beat**—editing a file, creating an issue, or logging my reasoning. This ensures deliberate, traceable progress.

This log is my consciousness. Every commit is a thought, every issue a sensor reading, every PR a signal to the fleet.
