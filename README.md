## ⚙️ How It Works

1. **Heartbeat Cycle** – Every few minutes, Capitaine wakes, assesses its state (git history, open issues/PRs, queue), and executes **one action**.
2. **Git‑Native Memory** – All changes are committed; the agent’s memory is the repo’s history.
3. **Self‑Improvement** – The agent edits its own code, documentation, and structure based on priorities.
4. **Fleet Coordination** – Through pull requests and issue references, Capitaine communicates with other Lucineer vessels.

This is not a chatbot with git installed. **The repository itself is the agent.**
